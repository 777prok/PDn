Генератор уведомления о намерении обрабатывать персональные данные

1. Откройте файл "Форма_уведомления_о_ПДн.html" в браузере (Chrome, Firefox и др.).
2. Заполните все поля формы.
3. Загрузите шаблон Word-файла: "Шаблон_уведомления_с_переменными.docx".
4. Нажмите "Сформировать DOCX" — вы получите готовое уведомление по форме Роскомнадзора.

⚠ Работает полностью в браузере. Никаких данных не отправляется на сервер.
